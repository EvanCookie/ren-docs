编译 / 构建 / 打包

Webpack

webpack 是一个用于现代 JavaScript 应用程序的 静态模块打包工具。

Vite

Vite 是一个超快速的前端构建工具，推动着下一代网络应用的发展

Rolldown （开发阶段）

Rolldown 是一个用 Rust 编写的 Js/Ts 打包器，旨在作为Vite未来使用的打包器。它提供了与 Rollup 兼容的 API 和插件接口，但在范围上与 esbuild 更相似。

Turbo 是一个针对 JS 和 TS 优化的增量捆绑器和构建系统，用 Rust 编写。

Babel

Babel 是一个 JavaScript 编译器，主要用于将 ECMAScript 2015+ 代码转换为当前和旧版浏览器或环境中的向后兼容版本的 JavaScript。

esbuild

一款速度极快的 Web 打包工具

Parcel

Parcel 零配置构建工具，将出色的开箱即用开发体验与可扩展的架构相结合，可以让您的项目从刚起步阶段发展为大规模生产应用程序。

SWC

SWC 是一个可扩展的 Rust 平台，适用于下一代快速开发工具。Next.js、Parcel 和 Deno 等工具以及 Vercel、字节跳动、腾讯、Shopify 等公司都在使用它。
